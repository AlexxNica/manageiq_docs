A provider in CloudForms is an external system that can be attached to CloudForms to be managed.

There are several types of providers, which include:

* Infrastructure providers
* Configuration management providers
* Automation management providers
* Cloud providers
* Networking management providers
* Middleware management providers
* Container providers
* Storage managers
