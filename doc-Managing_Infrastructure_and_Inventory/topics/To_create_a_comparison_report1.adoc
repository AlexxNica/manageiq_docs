= Creating a Cluster Comparison Report

Create a quick report of to compare clusters in CSV, TXT, or PDF formats.

. Create the comparison to analyze.
. Click  image:2107.png[] (*Download*).
. Click the output button for the type of report.
+
* Click  image:2133.png[] (*Download comparison report in TXT format*) for a text file.
* Click  image:2133.png[] (*Download comparison report in CSV format*) for a comma-separated file.
* Click  image:2134.png[] (*Download comparison report in PDF format*) for a PDF file.
